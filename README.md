## Modo de uso

#### Button

```html
<ngb-button [class]="['btn', 'btn-success']" (clickEvent)="sendToConsole($event)">hola mundo</ngb-button>
```



```javascript
sendToConsole(value: string) {
    console.log(value);
  }
```
